========== READ ME =============
 What is this application about?
-----------------------------------
The application enables a user to take a picture, and add text at the top and bottom to form a meme. The user will be able to share the photo on Facebook and Twitter, and by SMS or email.

 How can I use this application?
-----------------------------------
- To choose a photo and add text at the top and bottom.


How can I build this application?
-----------------------------------
1. Open Xcode.
2. Create a new project.
3. Design a storyboard an interface UIViewController.
4. Write the code.
5. run it!
